# 💜 Glassmorphism Login Form

Uma tela de login moderna, elegante e responsiva, desenvolvida com HTML e CSS puro. O projeto destaca o uso do efeito de **Glassmorphism** (vidro fosco), combinado com uma paleta de cores roxas vibrantes sobre um background imersivo.

## ✨ Funcionalidades

* **Design Moderno:** Interface de usuário limpa e atraente.
* **Efeito Glassmorphism:** Utilização de `backdrop-filter: blur()` e fundos semi-transparentes para criar a ilusão de vidro.
* **Ícones Integrados:** Uso da biblioteca [Boxicons](https://boxicons.com/) para ícones de usuário e cadeado.
* **Estilização Customizada:** Cores de texto, placeholders, links e botões perfeitamente alinhados na paleta roxa.
* **Responsividade:** Estrutura flexível baseada em Flexbox para manter o formulário centralizado em qualquer resolução.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do formulário.
* **CSS3:** Estilização, variáveis, Flexbox e efeitos visuais (blur, transparências e hover effects).
* **Boxicons:** Biblioteca de ícones.
* **Google Fonts:** Fonte 'Poppins' para uma tipografia limpa e moderna.
